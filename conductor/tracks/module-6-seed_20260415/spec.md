# Module 6 Curriculum Seed - Specification

## Context

Module 6 covers logarithms as inverses of exponential functions, logarithm properties, common and natural logarithms, and applied exponential/logarithmic models. The source curriculum lives in `curriculum/modules/module-6-logarithmic-functions.md` and lesson files `module-6-lesson-1` through `module-6-lesson-5`.

## Requirements

1. Treat `curriculum/modules/module-6-lesson-*` as canonical.
2. Create idempotent seed mutations for all five lessons.
3. Preserve logarithmic notation, base notation, inverse relationships, and applied contexts.
4. Identify activity-component gaps for logarithmic graphing or solving.
5. Add Module 6 standards and lesson-standard links once source mapping is confirmed.

## Acceptance Criteria

1. All five Module 6 lessons seed without duplicates.
2. Phase counts and order match curriculum headings.
3. Logarithmic properties and applied examples are represented faithfully.
4. Module 6 standards are seeded or explicitly documented as blocked by missing mapping.
5. `npm run lint` and relevant tests pass.

- New CAS-level logarithm solver work.
