# Project Sprint: Six Dataset Challenge

## Phase 1: Dataset Assignment + Task Framing
- **Goal:** Launch the differentiated group challenge.
- **Activity:** Each group receives one dataset from {{assets.group_datasets}} and confirms the shared deliverable: a balanced Balance Snapshot workbook built from new data, not the common class dataset from Lesson 7.

## Phase 2: Foundational-Proficiency How-To
- **Goal:** Ensure every group can complete the essential build moves.
- **Activity:** Use {{foundational_howto}} to walk groups through the non-negotiable sequence: classify accounts, confirm the equation, organize the Balance Sheet, and populate the chart inputs.

## Phase 3: Group Build
- **Goal:** Produce the first complete workbook draft.
- **Activity:** Teams build their own workbook from their assigned dataset. Every group must use the same product structure as Lesson 7, but the numbers and cases differ by dataset.

## Phase 4: Checkpoint + Reflection
- **Goal:** Confirm foundational completion before polish.
- **Activity:** Every group must show a balanced workbook draft, then record which build step still needs the most support before Lesson 9 polish.

---

## Teacher Notes
- **Pedagogy:** Lesson 8 is no longer generic work time. It is the differentiated dataset lesson and must stay tied to the same product structure established in Lesson 7.
- **Setup:** Pre-assign the six datasets so every class section can distribute group cases quickly.
