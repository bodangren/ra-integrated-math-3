# Unit 1 – Lesson Planning Matrix (Balance by Design)

## Lessons 1-7 Required Phase Rhythm

| Lesson | Intro / Hook | I Do | We Do | You Do | Exit Ticket | Reflection |
|---|---|---|---|---|---|---|
| L1 | Intro video, unit problem, familiar solutions, simulation | Teacher frames the unit challenge | Class explores simulation together | Students complete low-floor sort | Launch exit ticket | Why balance builds trust |
| L2 | Classification tension case | Teacher models account classification | Class sorts sample accounts | Students classify ledger rows | A/L/E exit ticket | Gray-zone accounts |
| L3 | Business-event hook | Teacher models dual-impact tracing | Class traces sample events | Students trace new events | Event-effects exit ticket | Surprising event patterns |
| L4 | Reporting-use hook | Teacher models Balance Sheet structure | Class maps accounts to sections | Students build mini report | Structure exit ticket | Readability and trust |
| L5 | Dirty-ledger trust hook | Teacher models Conditional Formatting rules | Class builds rules together | Students clean dirty ledger | Error-audit exit ticket | What red flags mattered |
| L6 | Prevention-vs-correction hook | Teacher models Data Validation controls | Class builds validation lists | Students lock down workbook | Validation exit ticket | Confidence in controlled data |
| L7 | Banker / audience hook | Teacher models the full scaffolded build | Class completes guided workbook steps | Students finish class snapshot | Snapshot exit ticket | What story the snapshot tells |

## Lessons 8-10 Project Expectations

| Lesson | Required Structure | Dataset Rule | Guidance Rule | Deliverable |
|---|---|---|---|---|
| L8 | New group build | Six groups receive six different datasets | Foundational-proficiency how-to required | Draft group workbook with balanced snapshot |
| L9 | Polish and advanced quality | Continue same Lesson 8 workbook | Polish / advanced-proficiency how-to required | Investor-ready workbook + script |
| L10 | Presentation to class | Present final group product | Presentation criteria must be explicit | live presentation + submitted workbook |

## Notes for Teacher Use

- Lesson 4 is intentionally accounting-led in Unit 1. The contract allows Lesson 4 to vary by unit, but Unit 1 needs one more accounting concept before the Excel build sequence begins.
- Lesson 7 is the common-class build that establishes the shared product expectations before differentiated group data begins.
- Lesson 8 must not collapse into generic "group work day" language during future redesign passes.
