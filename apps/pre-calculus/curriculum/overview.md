# AP Precalculus Curriculum

## Overview

4 units, ~54 lessons covering polynomial/rational functions, exponential/logarithmic functions, trigonometric/polar functions, and parametric/vectors/matrices.

---

## Unit 1: Polynomial and Rational Functions

*(Weeks 1–7)*

### Lessons / Topics

- 1.1 Change in Tandem
- 1.2 Rates of Change
- 1.3 Rates of Change in Linear and Quadratic Functions
- 1.4 Polynomial Functions and Rates of Change
- 1.5 Polynomial Functions and Complex Zeros
- 1.6 Polynomial Functions and End Behavior
- 1.7 Rational Functions and End Behavior
- 1.8 Rational Functions and Zeros
- 1.9 Rational Functions and Vertical Asymptotes
- 1.10 Rational Functions and Holes
- 1.11 Equivalent Representations of Polynomial and Rational Expressions
- 1.12 Transformations of Functions
- 1.13 Function Model Selection and Assumption Articulation
- 1.14 Function Model Construction and Application

### Core Objectives

- Analyze **rates of change** across function types
- Understand **polynomial structure** (zeros, end behavior, multiplicity)
- Analyze **rational functions** (asymptotes, discontinuities, holes)
- Translate between **multiple representations** of functions
- Build and justify **function models from real-world contexts**

---

## Unit 2: Exponential and Logarithmic Functions

*(Weeks 8–16)*

### Lessons / Topics

- 2.1 Arithmetic vs. Geometric Sequences
- 2.2 Linear vs. Exponential Change
- 2.3–2.4 Exponential Functions & Manipulation
- 2.5 Exponential Modeling
- 2.6 Model Validation
- 2.7 Composition of Functions
- 2.8 Inverse Functions
- 2.9–2.12 Logarithmic Expressions & Functions
- 2.13 Exponential/Logarithmic Equations & Inequalities
- 2.14 Logarithmic Modeling
- 2.15 Semi-log Plots

### Core Objectives

- Distinguish **linear vs exponential growth/decay**
- Work fluently with **exponential and logarithmic expressions**
- Understand **inverse relationships (exp ↔ log)**
- Solve **equations and inequalities** involving logs/exponentials
- Use functions for **data modeling and validation**
- Interpret **multiple representations** (graphical, algebraic, numerical)

---

## Unit 3: Trigonometric and Polar Functions

*(Weeks 17–23)*

### Lessons / Topics

- 3.1 Periodic Phenomena
- 3.2–3.3 Basic Trigonometric Functions
- 3.4–3.6 Graphs and Transformations
- 3.7 Sinusoidal Modeling
- 3.8 Tangent Function
- 3.9 Inverse Trigonometric Functions
- 3.10 Trigonometric Equations and Inequalities
- 3.11 Other Trig Functions (sec, csc, cot)
- 3.12 Equivalent Representations
- 3.13–3.15 Polar Coordinates and Functions

### Core Objectives

- Understand **periodic behavior** and trigonometric functions
- Graph and transform **sinusoidal functions**
- Solve **trigonometric equations**
- Model real-world phenomena (e.g., waves, cycles)
- Work with **polar coordinates and representations**
- Connect trigonometric representations across contexts

---

## Unit 4: Functions with Parameters, Vectors, and Matrices

*(Weeks 24–30)*

### Lessons / Topics

- 4.1 Parametric Functions
- 4.2 Parametric Motion Modeling
- 4.3 Rates of Change (Parametric)
- 4.4 Parametric Circles and Lines
- 4.5 Implicit Functions
- 4.6 Conic Sections
- 4.7 Parametrization
- 4.8 Vectors
- 4.9 Vector-Valued Functions
- 4.10–4.14 Matrices and Applications

### Core Objectives

- Represent motion using **parametric equations**
- Analyze **conic sections and implicit relationships**
- Perform operations with **vectors**
- Use **matrices for systems and transformations**
- Apply mathematics to **multi-dimensional modeling contexts**

---

## Big Picture Course Objectives

Across all units, students are expected to:

- Develop **procedural fluency** and symbolic manipulation
- Use **multiple representations** (graphs, tables, equations, words)
- Communicate reasoning using **mathematical language**
- Build, test, and justify **mathematical models**
- Use **technology (graphing calculators)** effectively
